# 🌟 Glimr: Real-Time AI-Powered Chat Ecosystem

Glimr is a high-performance, full-stack messaging platform built with a "Core AI" future in mind. It utilizes a monorepo architecture to share logic between a TypeScript backend and a React Native mobile application, eventually deploying to a scalable AWS infrastructure.

---

## 🎯 App Requirements

### Core Messaging

- **Real-Time Communication:** Sub-100ms latency for message delivery using Socket.io (WebSockets).
- **Cross-Platform:** Native iOS and Android experience via a single React Native codebase.
- **Persistence:** Reliable storage and retrieval of chat history and user profiles.

### AI & Intelligence (Phase 4)

- **Sentiment Engine:** Real-time emotional analysis of incoming and outgoing messages.
- **Smart Replies:** Context-aware suggested responses generated by LLMs.
- **Long-term Memory:** Vector-based retrieval of past conversations for AI context.

### Professional Infrastructure

- **Type Safety:** 100% TypeScript coverage shared across the entire stack.
- **Scalability:** Containerized microservices designed for cloud orchestration.
- **Automated DevOps:** CI/CD pipelines for automated testing and deployment.

---

## 💻 Tech Stack

### Frontend (Mobile)

- **Framework:** React Native (CLI)
- **Language:** TypeScript
- **State Management:** Zustand (High-performance, selective re-renders)
- **Navigation:** React Navigation
- **Real-Time:** Socket.io-client

### Backend (Server)

- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** TypeScript
- **Real-Time:** Socket.io
- **Database:** MongoDB Atlas (M0 Free Tier)
- **Containerization:** Docker

### Infrastructure & DevOps

- **Architecture:** Monorepo (npm Workspaces)
- **Cloud Provider:** AWS (EKS / ECS / App Runner)
- **IaC:** Terraform
- **CI/CD:** GitHub Actions
